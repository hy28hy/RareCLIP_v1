## 核心发现

### 1. No-bank 模式并没有显著降级性能
- 大部分数据集上，差异小于 1%
- 说明 Patch Rarity Branch (依赖记忆库) 并不是唯一关键分支

### 2. 记忆库在医学数据集上更有用
- Retina_RESC: auroc_sp 差异 -1.20
- 可能因为医学图像的正常样本更复杂，需要记忆库来建立正常特征分布

### 3. 息肉数据集不需要记忆库
- 所有息肉数据集的差异都接近 0
- 说明 Text Prompt Branch 已经足够强大
